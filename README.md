# Vue2-Order

> 基于Vue2实现的点餐Demo [在线地址](https://safaring.github.io/Vue2-Order/#/shop/1)

### 主页功能
- 店铺首页
- 餐品列表
- 购物车
- 餐品详情页
- 订单页

### 技术栈
vue2 + vue-router + vuex + rem + webpack

## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:5000
npm run dev

# build for production with minification
npm run build

```
